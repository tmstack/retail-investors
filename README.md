## 💡 为什么需要这个框架？

### 韭菜的三大幻觉

1. **"我能精准择时"** → 过度自信偏差，频繁交易侵蚀收益
2. **"这次真的不一样"** → 锚定效应，无视历史规律
3. **"市场在针对我"** → 外归因倾向，拒绝反思自身问题

### 真相

根据行为金融学研究（Kahneman & Tversky, 1979；Barber & Odean, 2000）：
- **交易最频繁的20%散户**，年化收益**低于市场7.1%**
- **散户平均亏损**：全球股市普遍现象
- **97%的交易者**长期来看是亏损的（Investopedia）

**韭菜不是智商低，韭菜是人性的受害者。**

## 效果示例

### 场景一：识别韭菜思维

> 输入：`每次买了就站岗，割了就拉升，好心痛`

```text
韭菜思维框架分析：

🔴 识别到的问题：
1. 羊群效应（追高）→ "买了就站岗"说明追高买入
2. 损失厌恶（死扛）→ "站岗"说明被套后不愿止损
3. 情绪化割肉 → "割了就拉升"说明恐慌底部割肉
4. 后见之明偏差 → "好心痛"说明事后后悔

💡 改进方案：
- 建立止损纪律：亏损10%强制止损
- 成本价解锚：问"如果现在是现金，我还会买吗？"
- 接受踏空：错过比亏钱好

🎯 你陷入的循环：
追高买入 → 被套站岗 → 恐慌割肉 → 立即拉升 → 后悔心痛
```

### 场景二：市场情绪分析

> 输入：`现在大家都在说牛市来了，我应该怎么办？`

```text
韭菜思维框架分析：

📊 市场情绪评估：
当所有人都谈论牛市时，往往是：
- Fear & Greed 指数 > 80（极度贪婪）
- 散户疯狂开户
- "这次不一样"流行

⚠️ 风险提示：
- 历史上，散户最乐观时往往是市场顶部
- 巴菲特："别人贪婪时恐惧"

💡 反向策略：
- 逐步减仓，锁定利润
- 保留现金，等待恐慌机会
- 不追热点，警惕泡沫

📌 检查清单：
[ ] 媒体全是好消息？
[ ] 所有人都在讨论股票？
[ ] 你有"不买就亏了"的感觉？
→ 如果是，可能需要冷静
```

### 场景三：操作建议

> 输入：`我看好的股票从50跌到20了，现在应该割肉还是补仓？`

```text
韭菜思维框架分析：

🚨 先检查你的思维陷阱：
- "从50跌到20" → 你还在用成本价思考（锚定效应）
- "割肉"这个词 → 说明你把止损当成失败（损失厌恶）

💡 成本价解锚测试：
不要想"我50买的"，问自己：
"如果我现在手握现金，没有任何持仓，20元我会买这只股票吗？"

├─ 会 → 可以持有或补仓，但要设止损
└─ 不会 → 说明你只是不愿承认错误，应该割肉

📋 决策树：
1. 基本面变差了吗？
   ├─ 是 → 立即止损
   └─ 否 → 进入第2步

2. 如果现在是现金，你会买吗？
   ├─ 会 → 持有，设止损线
   └─ 不会 → 止损

🎯 记住：
忘记成本价，只看未来价值。
```

## 功能特性

### 核心心智模型（7个）

| 模型 | 一句话描述 | 识别什么 |
|------|-----------|---------|
| **处置效应** | 卖盈持亏 | 为什么小赚频繁兑现，大亏不断累积 |
| **损失厌恶** | 损失痛苦是收益2.5倍 | 为什么不愿止损，小亏变大亏 |
| **羊群效应** | 追涨杀跌 | 为什么高买低卖 |
| **锚定效应** | 成本价执念 | 为什么"等回本就卖" |
| **过度自信** | 高估自己能力 | 为什么频繁交易 |
| **确认偏差** | 信息茧房 | 为什么越陷越深 |
| **情绪钟摆** | 贪婪恐惧摆动 | 什么时候反向操作 |

### 决策启发式（5条）

1. **情绪极端识别**：Fear&Greed < 20 恐慌买入，> 80 贪婪卖出
2. **成本价解锚**：忽略买入成本，基于未来价值决策
3. **反向清单验证**：热门标的反而谨慎
4. **止损纪律**：亏损10%强制止损
5. **仓位控制**：单只股票不超过20%

### 自查工具

- **每日快检**：交易前/中/后5秒检查清单
- **月度深度自查**：情绪决策、认知偏差、行为模式全覆盖
- **韭菜言论识别库**：120+典型韭菜言论特征

## 常见问题

**Q: 这个框架能帮我赚钱吗？**
A: 不能。框架只能帮你识别韭菜思维模式，避免常见错误。赚钱需要你自己的判断和纪律。

**Q: 反向策略一定有效吗？**
A: 不一定。"韭菜反着买"不是盲目与大众对立，而是基于理性判断和基本面分析。情绪极端可能持续很久。

**Q: 我如何知道自己是不是韭菜？**
A: 使用框架的"深度自查清单"，每月检查一次。诚实面对问题，是改进的第一步。

**Q: 止损和长期持有矛盾吗？**
A: 不矛盾。投机需要止损，投资可以长期持有。关键是区分"死守垃圾股"和"持有优质股"。

**Q: 为什么"买了就站岗，割了就拉升"？**
A: 这是因为陷入了"追高→被套→死扛→恐慌割肉→踏空"的韭菜循环。框架可以帮你打破这个循环。
